# Code Generator Agent – Codex CLI

This example demonstrates a **code generation agent** using the OpenAI Codex CLI.

## What it does

- Generates functions, classes, and modules from natural language descriptions
- Automatically creates unit tests
- Refactors existing code following best practices
- Uses sub-agents to generate and test in parallel

## How to use

```bash
cd examples/gerador-codex
codex
```

Or via command line:
```bash
codex run --memory=projects.json \
  --skill "code-generation" \
  "Implement a Queue class in Python with enqueue, dequeue, and peek."
```

## Structure

```
gerador-codex/
├── AGENTS.md          # This file (agent documentation)
├── codex-agent.ts     # TypeScript orchestration example
├── simple_agent.py    # Python agent example
├── projects.json      # Codex memory/context file
└── generated/         # Code generated by the agent (created during use)
```

Codex: 
def binary_search(data: list[int], target: int) -> int:
    low, high = 0, len(data) - 1
    while low <= high:
        mid = (low + high) // 2
        if data[mid] == target:
            return mid
        elif data[mid] < target:
            low = mid + 1
        else:
            high = mid - 1
    return -1
```
